### 3. NVIDIA OpenShell — Open-Source Agent Security Runtime (Level 0-1)
**Source:** https://nvidianews.nvidia.com/news/ai-agents — [vendor press release]
**Source:** https://awesomeagents.ai/news/nvidia-openshell-agent-sandbox-security/ — [practitioner analysis]
**Date:** 2026-03-16 to 2026-03-18 (GTC)
**Agent level:** Company (infrastructure)
**What:** NVIDIA launched Agent Toolkit at GTC with three components: (1) OpenShell — open-source Apache 2.0 sandbox runtime for AI agents. Could become the security layer beneath all platforms.
**Evidence level:** Level 0 (vendor announcement) + Level 1 (practitioner commentary)
**Key claims:**
- OpenShell: Apache 2.0, containerized, security-first
- 15+ enterprise partners including all major SaaS vendors
- Different approach from Microsoft Agent 365 governance or Anthropic Antspace sandboxing
- Still alpha/early-access
