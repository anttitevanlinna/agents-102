## How It Works

**Three research systems run continuously:**

1. **Platform watch** — semi-autonomous OODA cycles tracking Microsoft, Google, OpenAI, Anthropic, and vertical SaaS
2. **Domain research** — practitioners doing agentic work in operations, finance, HR, compliance, customer service, sales
3. **User signals** — your questions and comments steer future research (captured automatically)

**Start here:** `continuous-research/synthesis/index.md` routes you to the right topic file in one read.

## For Users

- Run `/research` to get oriented
- Ask any question — the system answers from evidence-grounded findings
- Your questions and comments are automatically captured as research steering signals
- Run `git pull` at the start of each session to get the latest research

## Branching

- `main` is the shared knowledge base
- Research agents running the OODA cycle system push directly to main
- User/experimental work goes on branches — Antti reviews and merges
