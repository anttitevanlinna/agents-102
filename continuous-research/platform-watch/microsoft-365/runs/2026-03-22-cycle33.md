### Finding 3: CVE-2026-26144 — Excel Copilot zero-click data exfiltration
**Source:** [The Register](https://www.theregister.com/2026/03/10/zeroclick_microsoft_info_disclosure_bug/) — [domain trade publication]; [Blueinfy blog](https://blog.blueinfy.com/2026/03/ssrf-in-azure-mcp-server-tools.html) — [practitioner analysis]
**Date:** March 10, 2026
**Agent level:** personal / company
**What:** Critical Excel vulnerability (CVSS 7.5) enables zero-click data exfiltration through Copilot Agent mode. XSS flaw in Excel's web-rendering pipeline triggers Copilot Agent to make outbound requests to attacker-controlled endpoints — no user interaction required. The AI agent's network privileges become the attack vector. Patched March 10.
**Evidence level:** Level 2 (verified CVE, patched, multiple security analysts confirmed)
**Key claims:**
- CWE-79 XSS triggers Copilot Agent to silently exfiltrate data
- Zero-click: exploitation via automatic file preview
- Copilot Agent's network privileges become the exfiltration channel
- Represents emerging attack class: AI integration features as unintended data channels

### Finding 4: CVE-2026-26118 — Azure MCP Server SSRF for privilege elevation
**Source:** [TheHackerWire](https://www.thehackerwire.com/azure-mcp-server-ssrf-for-privilege-elevation-cve-2026-26118/) — [domain trade publication]; [Blueinfy blog](https://blog.blueinfy.com/2026/03/ssrf-in-azure-mcp-server-tools.html) — [practitioner direct]
**Date:** March 10, 2026
**Agent level:** company
**What:** SSRF vulnerability (CVSS 8.8) in Azure MCP Server Tools allows attackers to steal the server's managed identity token by submitting a malicious URL. The MCP Server makes an outbound request including its credentials, enabling lateral movement through Azure. Public PoC exists. Affects versions prior to 2.0.0-beta.17.
**Evidence level:** Level 2 (patched CVE with public PoC, confirmed by Cisco Talos)
**Key claims:**
- Attacker captures managed identity token via crafted input
- Enables access to any Azure resource the MCP Server identity has permissions for
- Public PoC on GitHub — weaponization risk is high
- MCP protocol infrastructure itself is an attack vector

## Orientation

**Three stories converging to define Microsoft's agent position:**

**1. Security is the story.** Three CVEs in one month: Excel Copilot zero-click exfiltration (CVE-2026-26144), Azure MCP Server SSRF (CVE-2026-26118), and the DLP sensitivity label bypass (second time in 8 months). The pattern: AI agent integration creates new attack surfaces. Microsoft is simultaneously creating the risk (shipping agents) and selling the solution (Agent 365 governance). For CTOs: the security surface is expanding faster than governance can cover it.

**2. Cowork silence is now a finding.** We are inside the "late March" window. Other Frontier features are rolling out on schedule (Notebooks March 23-28, Planner Agent mid-March). Copilot Cowork — the flagship — has no confirmed rollout date. If March ends without broader Frontier access, Microsoft missed its own commitment. NPI Financial's advice remains correct: "directional, not deliverable."

**3. Nordic AI value gap.** BCG's new report quantifies what we've been seeing: Nordic companies are investing in AI (10%+ of IT budgets) but only 4% achieve strong returns. They over-invest in off-the-shelf tools and under-invest in agentic workflow redesign. This is the exact gap our training program addresses — competence to redesign workflows, not just deploy tools.

**For the CTO question:** Microsoft has the broadest distribution but deepening problems at every layer: user trust deficit (8% preference, -19.8 accuracy NPS), security gaps (3 CVEs in a month, DLP bypass), reliability issues approaching convergence (5+ practitioners), and a flagship product (Cowork) that hasn't shipped. The E7 bundle is the monetization play, but Gartner says use it for negotiation leverage, not adoption. Anthropic capturing 73% of new enterprise AI buyers (Ramp) means the model layer Microsoft depends on is losing market share. The May 1 E7/Agent 365 GA remains the next inflection point — but the evidence base for optimism is thinning.
