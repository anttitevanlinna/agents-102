### 6. Agent Identity Crisis — 93% Unscoped
**Source:** [AGAT Software: AI Agent Security 2026](https://agatsoftware.com/blog/ai-agent-security-enterprise-2026/) — [practitioner analysis]; [Grantex: State of Agent Security](https://grantex.dev/report/state-of-agent-security-2026) — [practitioner direct]
**Date:** 2026
**Agent level:** Company (governance/security)
**What:** Audit of 30 AI agent projects: 93% use unscoped API keys, 0% assign cryptographically verifiable agent identity, only 13% include action logging, only 21.9% treat agents as identity-bearing entities. Grantex (Apache 2.0) proposes signed JWT grant tokens, per-agent DIDs, FIDO2/WebAuthn consent flows, granular revocation, immutable audit trails. IETF Internet-Draft submitted. OpenAI has no agent identity management comparable to Microsoft Entra. OpenAI's response: "Codex Security" (March 6) scans commits for vulnerabilities — a scanning tool, not an identity/RBAC solution.
**Evidence level:** Level 2-3 (multiple independent sources converge on same security gaps; Grantex emerging as potential standard but adoption unknown)
**Key claims:**
- 93% of agent projects use unscoped API keys (AGAT audit)
- 0% assign unique agent identity (AGAT audit)
- Grantex proposing IETF standard for agent identity
- OpenAI has no agent identity management system

## Orientation

The story is sharpening: **OpenAI is losing enterprise ground to Anthropic faster than previously measured.** Ramp's March data (70%+ of new buyers choosing Anthropic) is the most significant new finding — this is real spending data, not surveys. The market share reversal is accelerating, not stabilizing.

Meanwhile, OpenAI's product strategy is fragmenting: consumer super-app + desktop merge + Frontier + IPO prep + consulting partnerships, all simultaneously. The EEA connector situation is improving but inconsistently — feature-flag rollout suggests infrastructure immaturity rather than strategic choice.

The hybrid computer-use architecture (TinyFish/Stagehand/Skyvern) is now convergence-level evidence — this is the production pattern for computer use, regardless of which model backs it. GPT-5.4 leads benchmarks but pure-AI computer use remains "impressive but unreliable" in the real world.

For the CTO question: OpenAI remains the most widely adopted (900M users) but the enterprise evidence gap vs. Anthropic is widening. The announcement-to-deployment gap is the story. Frontier at 7+ weeks with zero evidence is becoming a red flag, not a timing issue.
