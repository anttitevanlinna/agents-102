# Building Guardrails (and Skills)

A reference document that grows across the training. Each module that adds a new twist on guardrails or skills adds the next section. "Guardrails" isn't one thing — it's the same pattern scaled up across modules.

**Status:** outline only. Sections fill in as each module is written.
