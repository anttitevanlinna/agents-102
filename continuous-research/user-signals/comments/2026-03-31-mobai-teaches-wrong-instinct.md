Practitioner critique of MobAI: it's fundamentally limited because it teaches the wrong mental model — "human is alert and catches issues." This works at small scale but actively trains the wrong instinct for scaled AI work, where humans CANNOT watch in real-time because output volume exceeds attention capacity.

Four-phase progression identified:
1. CollabAI/MobAI — human watches AI, catches mistakes live (breaks when output exceeds attention)
2. Compound Engineering — solo + AI, review own output (breaks when you need more than one judgment)
3. SwarmAI — parallel generation, group evaluates (breaks when volume overwhelms the group)
4. Scaled AI — design verification systems, agents check agents (the actual target state)

MobAI is a valid Phase 1 on-ramp but should not be presented as the end state. The real skill is Phase 4: designing the systems that watch for you, not watching yourself.
