  - Intuit: TurboTax tax preparation agents

### GPT-5.4 — Native Computer Use + Strong Benchmarks
Released March 5, 2026. Most significant model release for agentic work.
- **Native computer use** — first OpenAI model with desktop interaction (keystrokes, mouse, menus)
- **Tool search** — lightweight tool list, on-demand lookup. 47% token reduction, same accuracy across 250 tasks/36 MCP servers
- **GDPval:** 83% on professional knowledge work (44 occupations), up from 70.9% (GPT-5.2). 33% fewer false claims.
- **OSWorld:** 75% (above 72.4% human average for software navigation)
- **vs Opus 4.6:** GPT-5.4 leads on knowledge work (83% GDPval) and computer use (75% OSWorld). Opus 4.6 leads on coding (80.8% SWE-Bench) and web research (84% BrowseComp).
  **Source:** [Fortune](https://fortune.com/2026/03/05/openai-new-model-gpt5-4-enterprise-agentic-anthropic/) — [general press]; [BuildFastWithAI](https://www.buildfastwithai.com/blogs/gpt-5-4-review-benchmarks-2026) — [practitioner analysis]
  **Date:** March 2026
  **Evidence level:** Level 2. Independent reviews confirm improvements. Enterprise deployment evidence thin (16 days old).

### AgentKit — Mixed Practitioner Reviews, Developer-Only
- Visual Agent Builder (beta), Connector Registry (beta), ChatKit (GA), eval tools (GA).
- Praised for speed of building simple agents. Criticized: OpenAI-model-only, limited retrieval, requires backend setup.
- Viewed as complementary to N8N/Zapier, not replacement.
  **Source:** [Unite.AI](https://www.unite.ai/agentkit-by-openai-review/) — [practitioner analysis]
  **Date:** 2026
  **Evidence level:** Level 2. Independent reviews exist. No business user (non-developer) deployments found.

### Codex Skills — Acknowledged Not for Business Users
- Skills now include project management (Linear triage), cloud deployment, image generation, automations.
- **Critical limitation from OpenAI ecosystem:** "It was built by engineers, for engineers... Getting started requires running commands like npm install from a terminal, which is a complete non-starter for someone in customer support or IT."
  **Source:** [eesel.ai](https://www.eesel.ai/blog/openai-codex) — [practitioner analysis]
  **Date:** 2026
  **Evidence level:** Level 1. Codex Skills for non-coding business work is aspiration, not shipped capability.

### Open Responses — Credible Backing, No Deployment Evidence
- Co-developed with Hugging Face. Backed by OpenRouter, Vercel, LM Studio, Ollama, vLLM.
- Replaces deprecated Assistants API (sunset Aug 26, 2026).
- Supports MCP tools natively.
- Anthropic and Google have NOT joined — betting on own formats.
  **Source:** [InfoQ](https://www.infoq.com/news/2026/02/openai-open-responses/) — [domain trade publication]; [Hugging Face](https://huggingface.co/blog/open-responses) — [practitioner direct]
  **Date:** February 2026
  **Evidence level:** Level 1-2. Infrastructure standard, not business user feature.

### Write-Action Connectors — Shipped, Zero Adoption Evidence
- Write actions for Outlook (draft emails), Google Docs/Sheets (create), calendar (book meetings). Disabled by default.
- Atlassian Rovo MCP connector: Jira + Confluence + Compass with Jira WRITE actions.
  **Source:** [OpenAI Help Center](https://help.openai.com/en/articles/11391654-chatgpt-business-release-notes) — [vendor documentation]; [gecco](https://www.geccohq.com/news/chatgpt-can-now-act-inside-outlook-google-docs-and-your-calendar) — [tech press]
  **Date:** March 2026
  **Evidence level:** Level 1. Features exist. Zero practitioner reports of production use.

### Recurring Task Scheduling — Early Beta, No Adoption Evidence
- Schedule one-off or recurring tasks (daily, weekly, monthly). Limit: 10 active tasks/user.
- Connectors enable tasks acting on Outlook, Docs, Sheets, calendars.
  **Source:** [OpenAI Help Center](https://help.openai.com/en/articles/10291617-scheduled-tasks-in-chatgpt) — [vendor documentation]
  **Date:** 2026
  **Evidence level:** Level 1. Documented capability. Zero business user reports found.

## Orientation
GPT-5.4 is the most significant new development — native computer use puts OpenAI on par with Claude Computer Use for legacy system integration potential. The Frontier enterprise story remains unverified 6+ weeks after launch. Write-action connectors and recurring tasks are the closest to "real business agent work" that's actually shipped and accessible, but adoption evidence is absent. The gap between announcement and deployment continues to widen.
