### Enterprise Deployment Claims (All Level 0)
Four companies named in AWS materials: Epsilon, Ericsson, Southwest Airlines, PwC
- All four sourced from AWS blog posts or partner co-marketing pages
- PwC page returned 404 on April 17, 2026
- Ericsson's own press (Computer Weekly, rcrwireless, intelligentcio.com) references Mistral AI and NetCloud — no AgentCore mention
- Zero independent practitioner accounts found after exhaustive search

### SailPoint Integration
- Products available on AWS Marketplace (purchasing path clear)
- Technical integration: agent identity governance, lifecycle, least-privilege via CloudTrail
- Implementation reality: 12+ months to maturity, professional services cost 3x software price (from independent SailPoint reviews)
- Status: announced partnership with marketplace listing; no production deployment evidence

### CVE-2026-4269 — RESOLVED
- Fixed in v0.1.13
- Only affected users who built Starter Toolkit between Sept 24, 2025 and fix date
- CVSS 7.5 (High). S3 bucket squatting — attacker registers predictable S3 bucket name, injects malicious payload during build
- Source: GitHub Security Advisory (authoritative)

### Lock-in Risk — New Independent Assessment
- Kai Waehner (independent practitioner, April 6, 2026): places AgentCore in "Risky and Captured" quadrant
- Key argument: multi-model flexibility sits above a proprietary orchestration layer (memory, sessions, governance, observability) that creates compounding structural dependency
- This is the strongest independent critique of AgentCore found to date

- AWS data center expansion in Nordics is happening (infrastructure) but no Nordic AgentCore agents found
