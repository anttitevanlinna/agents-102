### 1. Computer Use Wars Crystallize — Three Architectures, First Practitioner Comparisons

**Sources:**
- [AIble w/ My Mind: Perplexity Computer vs Claude Code vs Cowork vs Manus — Tested Side by Side](https://aiblewmymind.substack.com/p/perplexity-computer-vs-claude-code-cowork-manus-comparison) — [practitioner direct]
- [AI Agent Choices: Claude Computer Use Review 2026](https://aiagentchoices.com/agents/claude-computer-use/) — [practitioner analysis]
- [Brainroad: Claude Computer Use — AI Agent Desktop Control 2026](https://brainroad.com/claude-computer-use-what-it-means-for-ai-agents-in-2026/) — [practitioner analysis]
- [Erkan's Field Diary: Claude vs Perplexity Computer Tasks](https://erkansaka.net/2026/03/16/claude-vs-perplexity-comparison/) — [practitioner analysis]
- [TrySliq: Perplexity Computer vs Claude Cowork](https://www.trysliq.com/blog/perplexity-computer-vs-claude-cowork) — [practitioner analysis]

**Date:** March 2026
**What's NEW (not previously captured):**

**First head-to-head practitioner comparison published.** AIble w/ My Mind (Substack) ran a side-by-side test of Perplexity Computer, Claude Code, Claude Cowork, and Manus AI on two real tasks (real estate property dossier + live AI news app). Evaluation method: four independent AI models (GPT 5.1, Gemini 3 Pro, Claude Sonnet 4.5, Grok 4) scored anonymized outputs.

**Results:**
- **Perplexity Computer:** Won both tasks. Only tool to get zoning code (R6A) correct with deep links to primary sources. Research-anchored, real-time web access. Cost: ~$18 and ~$8 per task (credit-based at $200/mo).
- **Claude Code:** 2nd on property research, 3rd on news app. Best-looking apps, solid executive summaries. But: "catastrophic hallucination" with 2026 dates making content unreliable.
- **Claude Cowork:** 3rd on property research, 2nd on news app. Competent structure, clean design. Lost credibility by guessing zoning data with "Likely" — problematic for professional contexts.
- **Manus AI:** Last on both. Hallucinated irrelevant information. Generic news with broken links.

**Three architectures confirmed:**
1. **Anthropic (local machine):** Permission-first, runs on your desktop, data stays local. Pro $20/mo, Max $100-200/mo.
2. **Perplexity (cloud orchestrator):** 19 AI models, multi-agent routing, runs in cloud. $200/mo Max only.
3. **OpenAI (sandboxed):** Cloud sandbox, separate virtual computer environment.

**Critical finding for Perplexity:** Without Claude as its reasoning backbone, Perplexity Computer's orchestration layer loses its primary brain. Perplexity is one of Anthropic's biggest customers — validates Claude's value by choosing it over GPT or Gemini for reasoning.

**Claude Computer Use practitioner verdict:** "Groundbreaking technology for automating legacy systems. Not yet ready for production environments." Business readiness rated 3/10, beginner-friendliness 2/10. Successfully handles desktop app control, legacy system form-filling, data extraction from non-API sources, and multi-application workflows. But: error-prone, massive token consumption on complex tasks, requires technical setup.

**New technical detail:** "Zoom Action" feature added to Computer Use API — lets Claude inspect small UI elements at high resolution before clicking, fixing a key reliability problem from early beta.

**Evidence level:** Level 2 (multiple independent practitioner tests, specific results, but still early days — 3 days since launch)

**Agent level:** Personal (all computer use agents are personal-level today — no team/company deployment path exists)
