### Finding 2: Microsoft Build 2026 — Project Polaris replaces OpenAI in GitHub Copilot; Windows becomes agent OS

**Date:** June 2, 2026
**Agent level:** company
**What:** Microsoft Build 2026 (June 2–3) produced three major strategic moves:

**(A) Project Polaris — Microsoft's own coding model replaces OpenAI in GitHub Copilot (August 2026):**
- Mixture-of-experts architecture, outperforms GPT-4 Turbo on coding benchmarks (notably Rust and Haskell)
- Runs on Microsoft's custom Maia AI accelerators in Azure
- "Microsoft now controls the model, the inference infrastructure, and the developer experience end-to-end"
- MAI-Thinking-1 also announced: first Microsoft reasoning model, 35B active parameters, 97% on AMI2025 benchmark, trained without OpenAI data
- MAI-Code-1-Flash: 5B-parameter coding model for VS Code

**(B) Windows as agent platform — three-layer architecture:**
- **Windows Agent Framework (WAF) v1.0**: MIT-licensed SDK, agents defined in YAML, runs across local Windows machines, Windows 365 Cloud PCs, Azure Arc devices
- **Windows Agent Runtime**: OS-level native APIs — agents as first-class OS citizens (not just application processes)
- **Windows Agent Store**: Curated marketplace, 85% revenue share for agent creators
- **Azure Agent Mesh**: Federated agent execution across on-premises/Cloud PCs/edge devices, Q4 2026 GA

**(C) Multi-model Copilot — Claude officially first-party in M365 and Azure Foundry:**
- Azure AI Foundry: 12,000+ models including OpenAI, Anthropic (Claude Opus 4.8), MAI, Llama, Mistral
- Claude available in M365 Copilot mainline chat via Frontier program, Copilot Studio, Excel Agent Mode, Copilot Researcher
- Microsoft Foundry: Claude is routable alongside OpenAI and open-source in single Entra ID + Purview-governed surface
- GitHub Copilot App: Desktop agent IDE supporting OpenAI, Anthropic, and Google models with multi-session management
- MCP server support for Copilot agents: shared tool surface with Claude and Gemini agents

**(D) Other enterprise infrastructure:**
- **Microsoft Scout**: Always-on autonomous agent across Teams, Outlook, OneDrive, SharePoint with Entra identity governance
- **Azure HorizonDB**: PostgreSQL service optimized for agents — 3x faster transactions, vector indexing, in-database model access
- **MDASH**: Multi-model agentic security system using 100+ agents to identify exploitable bugs
- **Microsoft Execution Containers (MXC)**: OS-native isolation with granular policy controls
- **Web IQ**: Model-agnostic, MCP-native internet access layer for agents
- **Microsoft Discovery** (GA): Automated scientific method loop (agents + HPC + knowledge graphs + lab automation)
- **Project Solara**: Chip-to-cloud platform for agent-first devices including desk devices (MediaTek) and wearable badges (Qualcomm)

**(E) Customer evidence (partial, vendor-surfaced):**

- Land O'Lakes demo: 90% accuracy on butter report generation (demo, Level 0)
- Mayo Clinic partnership: building healthcare-specific frontier model (announced, Level 0)
- AccuWeather, Best Buy, Target exploring Project Solara devices (exploring, not deployed)
