## HEADLINE FINDINGS

### 1. Computer Use Wars — First Practitioner Head-to-Head Published

The first independent side-by-side test of four computer-use/agentic tools has been published (AIble w/ My Mind, Substack). Two real tasks tested: real estate property dossier and live AI news app. Four AI models evaluated anonymized outputs.

**Rankings:**
1. **Perplexity Computer** — won both tasks on accuracy and depth. Only tool to get zoning code correct with primary source links. Cost: ~$18 and ~$8 per task.
2. **Claude Code** — solid executive summaries, best-looking apps. "Catastrophic hallucination" with dates.
3. **Claude Cowork** — competent structure. Lost credibility by hedging with "Likely" on verifiable data.
4. **Manus AI** — last. Hallucinated irrelevant information, broken links.

**Critical dependency finding:** Perplexity Computer uses Claude as its primary reasoning backbone. Without Claude, the orchestration layer loses its brain. Anthropic wins either way — directly via Claude products, or indirectly via Perplexity's dependency.

**Claude Computer Use practitioner verdict:** "Groundbreaking for legacy systems. Not production-ready." Business readiness: 3/10. New "Zoom Action" feature improves UI element precision.

**Evidence level:** Level 2 (multiple practitioner tests, still day 2-3)
