### Finding 1: Foundry Agent Service reaches GA with private networking (March 16, 2026)
**Source:** [Microsoft Foundry Blog](https://devblogs.microsoft.com/foundry/foundry-agent-service-ga/) — [vendor blog, Level 0]
**Also:** [Redmondmag](https://redmondmag.com/articles/2026/03/17/microsoft-brings-production-ready-ai-agents-at-gtc.aspx) — [tech press]
**Date:** March 16, 2026
**Agent level:** Company
**What:** Foundry Agent Service is now GA with: (1) end-to-end private networking (BYO VNet, no public egress), (2) MCP authentication (key-based, Entra, OAuth passthrough), (3) production-grade evaluations (coherence, relevance, groundedness + custom evaluators), (4) built on OpenAI Responses API (wire-compatible). Three agent types: Prompt (GA, no-code), Workflow (GA, visual/YAML), Hosted (still Preview). Voice Live API in preview. Multi-model: Claude Opus 4.6, Sonnet 4.6, GPT-Realtime-1.5, Grok 4.0, NVIDIA Nemotron all available.
**Evidence level:** Level 0 (vendor announcement). No independent deployment evidence in the GA blog itself.
**Key claims:**
- Private networking = no public egress, container/subnet injection
- Wire-compatible with Responses API (minimal migration)
- Hosted agents still Preview; billing starts no earlier than April 1
- Named customer: Corvus Energy (marine battery systems, 1,500 vessels) — using Foundry + Fabric for inspection workflows. Quote from VP Global IT. But: no specific metrics, no independent verification.

- "Tens of millions" in customer registries in 2 months
- 29% of agents operate without IT approval (Microsoft Cyber Pulse report)

- Shadow agent quarantine capability
- Launches May 1, 2026 at $15/user/month (standalone) or part of E7 ($99/user)

## Orientation

Microsoft's Wave 3 / GTC announcements represent the most significant infrastructure buildout of any vendor:
- **Foundry GA** solves real problems (private networking, multi-model, evaluations) for developers
- **Agent 365** is the first governance product targeting shadow agents — a real enterprise need
- **CUA multi-model** (OpenAI + Claude) is unique and practically useful
- **E7 pricing** becomes more attractive as E3/E5 prices increase July 1

**But the pattern holds: infrastructure ahead of adoption.** Every announcement is vendor-level. Zero independent deployment evidence for any new feature. Copilot Cowork — supposedly the showcase — still has zero user reviews 12 days after broader access was promised. Agent 365 finds 500K "agents" at Microsoft (likely includes any automated process) but no external enterprise has reported similar discovery.

**For the CTO question:** Microsoft's governance story (Agent 365) is now the strongest differentiator. The shadow agent problem is real and independently confirmed. If a CTO's concern is "we have agents we don't know about" — Microsoft is the only vendor with a product response. But the actual agent capabilities (Cowork, CUA, Foundry) remain unproven in production.
