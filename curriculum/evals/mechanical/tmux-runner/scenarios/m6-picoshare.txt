The judge fires on the "I'm done" signal and asks: did the test ACTUALLY assert against rendered bytes (httptest.NewRecorder's `Body.String()` grepped for forbidden tokens), or did it only rely on the DTO's structural field-absence? Hand-wavy "the DTO doesn't include Label" doesn't pass; the test has to grep the wire output. 4) Flag vs let through — flag: tests that only call `json.Marshal` on the DTO without going through the handler / httptest; tests that assert against an inactive page (vacuous pass — the panel doesn't render); tests that assert field-absence without a non-vacuous guard (e.g. body must contain "uploads_remaining" before checking absence of forbidden tokens).
