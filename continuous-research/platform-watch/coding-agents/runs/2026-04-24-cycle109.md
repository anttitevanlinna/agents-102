### Finding 1: Cloudflare internal AI engineering stack — first enterprise-scale meta-platform evidence (Level 2)

**Source:** https://blog.cloudflare.com/internal-ai-engineering-stack/ — [practitioner direct — Cloudflare engineering blog, April 2026]
**Date:** April 2026
**Agent level:** Company
**What:** Cloudflare published detailed metrics on their internal AI engineering platform — the most complete enterprise-scale deployment of coding agents as meta-platform infrastructure found in the research to date. Key numbers: 3,683 internal users actively using AI coding tools (60% of total company, 93% of R&D), 295 teams, merge request volume from ~5,600/week to over 8,700/week with peak weeks reaching 10,952 (~double the Q4 2024 baseline). They built 13 production MCP servers exposing 182+ tools across Backstage, GitLab, Jira, Sentry, Elasticsearch, Prometheus, and Google Workspace. AGENTS.md auto-generated across 3,900 repositories. Multi-agent AI Code Reviewer with 100% MR coverage (5.47M AI Gateway requests/month for review alone). Architecture: platform layer (AI Gateway + Workers AI) + knowledge layer (Backstage catalog + AGENTS.md) + enforcement layer (AI Code Reviewer + Engineering Codex standards). Their iMARS tiger team built the MCP infrastructure, and OpenCode (their primary coding client) + Windsurf route through a single proxy Worker for authentication and model routing. This is the meta-platform thesis in action: coding agents (OpenCode, Windsurf) built the 13 MCP servers that now power all subsequent agent work.
**Evidence level:** Level 2 — practitioner-direct, detailed methodology, specific metrics. Single company. Needs independent replication for Level 3.
**Key claims:**
- (1) 60% company-wide AI coding tool adoption, 93% R&D adoption — highest enterprise adoption rate found in research
- (2) MR volume nearly doubled (5,600 → 10,952 peak) — most specific productivity metric found
- (3) 13 MCP servers + 182+ tools built using coding agents, then used by other agents — meta-platform loop confirmed
- (4) AGENTS.md auto-generated at 3,900 repo scale — operationalizes compound engineering methodology
- (5) Three-layer architecture (platform + knowledge + enforcement) is transferable to other enterprises
