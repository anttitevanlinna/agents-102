**Agent level:** team (API-level developer tooling)

Realistic migration effort: 1-4 engineering weeks for a small production system; enterprise-scale migration is 9-12 months (per SocialCrawl analysis). **Critical parallel track:** Microsoft is simultaneously retiring the Azure OpenAI Assistants API on August 26, pushing Azure-hosted deployments to Foundry Agents — a different migration path from OpenAI's Responses API route. - No automated migration tool confirmed

- docx support dropped (practitioners report this as a blocker)
- State management shift requires explicit developer management of tool loops
- Thread backfill: manual or accept data loss
