### Finding 1: Code with Claude 2026 (May 6) — Dreaming, Outcomes, Multi-Agent Orchestration Ship for Managed Agents
**Source:** https://claude.com/blog/new-in-claude-managed-agents — [vendor press release]; https://simonwillison.net/2026/May/6/code-w-claude-2026/ — [practitioner direct]; https://www.technologyreview.com/2026/05/21/1137735/anthropics-code-with-claude-showed-off-codings-future-whether-you-like-it-or-not/ — [general press]; https://letsdatascience.com/blog/anthropic-dreaming-claude-managed-agents-self-improving-may-6 — [domain trade publication]
**Date:** May 6, 2026
**Agent level:** Company (Managed Agents runtime)
**What:** Anthropic held "Code with Claude 2026" developer event May 6, shipping three new Managed Agents capabilities. **Dreaming:** agents review prior sessions overnight, identify gaps, write new memory entries that subsequent sessions use — documented self-improvement without model retraining. **Outcomes:** builder writes a rubric describing success; a separate grading agent (which has not seen the task agent's reasoning) scores the output and kicks it back for another run if it fails — explicit eval loop baked into the runtime. **Multi-agent orchestration:** fleet of specialist agents running in parallel, coordinated by a lead agent. Additional: Claude Code rate limits doubled across Pro/Max/Team/Enterprise; peak-hour throttle lifted. SpaceX Colossus deal (300+ megawatts of compute) disclosed as capacity expansion. Strategic signal: Anthropic chose not to ship a new model at a developer event — the message is that orchestration infrastructure is the next competitive axis, not raw model capability.
**Evidence level:** Level 1 for enterprise deployment outcomes (too new); Level 2 for the Outcomes quality mechanism (vendor internal testing: +10pp output quality from grading loop, disclosed in Anthropic Cookbook — vendor-internal benchmark)
**Key claims:**
- Dreaming: scheduled overnight session analysis → self-written memory entries → next session uses them. Harvey (legal AI) pilot: task completion rates ~6x (vendor-event report, Level 1)
- Wisedocs (medical documents): review time cut 50% using Dreaming + Outcomes (vendor-event report, Level 1)
- Outcomes: grading agent scores output against rubric; +10pp quality improvement in Anthropic internal benchmarks (vendor-internal, not independently verified)
- Rate limits doubled on all paid plans; peak-hour throttle lifted entirely
- SpaceX Colossus: 300+ megawatts — the compute deal was disclosed because capacity had been the growth constraint (API volume growing faster than projected)
- Mercado Libre: 23,000 engineers, targeting 90% autonomous coding by Q3 2026 (vendor-event showcase — directional, not independently verified)
- Boris Cherny quote: "The default isn't 'I'm going to prompt Claude' — the default is now 'I'm going to have Claude prompt itself'"

### Finding 7: Nordic Signal — Deloitte State of AI in Nordics 2026 — Agentic AI 3+ Years Away for Half of Nordic Orgs
**Source:** https://www.deloitte.com/fi/fi/Industries/technology/research/state-of-ai-in-the-nordics.html — [practitioner analysis / consulting firm research]
**Date:** 2026 (Deloitte Finland, published April–May 2026)
**Agent level:** Company
**What:** Deloitte Nordic AI report surveyed 170 senior Nordic executives (Denmark, Finland, Norway, Sweden) benchmarked against 3,235 global respondents. Key agentic AI findings: only 5% of Nordic organizations report high expertise in agentic AI; 49% expect agentic AI transformation to be more than 3 years away; strategic readiness dropped sharply from 61% to 43% year-over-year (not improving — declining despite increased AI awareness). Talent preparedness at only 14%. Positive signals: share of Nordic orgs where 40%+ employees have access to approved AI tools rose from 37% → 56% in one year. Only 18% achieve revenue growth from AI; 20% have appointed accountability for AI value outcomes. 84% cite data privacy/security as concerns; 67% prioritize security investments. No named platform preferences in accessible data.
**Evidence level:** Level 2 (170 executive survey, independent consulting firm — reasonable sample for senior-executive Nordic view, not general workforce)
**Key claims:**
- Only 5% Nordic orgs with high agentic AI expertise (N=170 senior executives)
- 49% expect agentic transformation 3+ years away — confirms competence-first thesis; Nordic orgs are not ready to choose platforms
- Strategic readiness declined 61% → 43% — counterintuitive: AI awareness up, strategic readiness down (overwhelm?)
- Tool access spreading fast: 37% → 56% of Nordic orgs with 40%+ employee access in one year
- Only 18% achieving revenue growth from AI — execution gap is real and quantified in Nordics
- Confirms sustained gap: Nordic orgs are in experimentation, not deployment, for agents

## Orientation

The competitive frame has shifted from "which model" to "which agent runtime." Anthropic (Dreaming/Outcomes/Orchestration via Managed Agents), Microsoft (Agent 365 + agent telemetry), OpenAI (Workspace Agents), and Salesforce (Agentforce Operations) all shipped or GA'd runtime/orchestration infrastructure in the April 29–May 6 window. The race is now for who owns the layer that runs agents at organizational scale — not who has the best single model. Anthropic's strategic choice to skip a new model at a developer conference and instead ship runtime features is the clearest signal of this shift. Nordic data confirms the competence gap is the first constraint to solve before any of this matters in that market.
