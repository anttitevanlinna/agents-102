# Cowork tour — pre-work for Claude Basics

A plain prompt file for use during pre-work. The participant pastes this into a fresh Cowork session connected to a safe personal folder. Claude walks them through the surface area they will use in the live crux exercise and homework.

## When to use

- Pre-work before the cohort, ~15 minutes, mandatory
- Pre-cohort dry-run by the organisers against a safe personal folder and the agreed shared synthesis location

## The prompt

```
This is my first time using Cowork properly. Walk me through what I'm looking at, in chat, one stop at a time. Ask me to confirm each stop before moving to the next.

The five stops:

1. The connected folder. Name the folder I'm connected to right now and what it means for files saved here. One sentence.

2. The chat input. Where I type prompts. Anything special I should know about pasting multi-line content.

3. The scrollback. How long it persists, what survives a refresh, what doesn't. One short note.

4. Save and read primitives. When I ask you to "save X as Y.md" or "read Z", what actually happens on disk. One short note.

5. Connectors. Whether O365, Jira, or anything else is wired into this session. If yes, name what; if no, say so.

After all five stops, summarize in 3 lines: where my files go, what survives between sessions, what to do if I want a fresh session vs. continue this one. Save the summary as cowork-tour-notes.md in this folder.
```

## Notes

- The five stops are deliberately short. The point is exposure, not training. The participant arrives at Module 1 having seen the surface area once
- The connector list (stop 5) is informational. The cohort's actual connector setup is the organisers' pre-cohort coordination, not the participant's pre-work
- The summary saved to `cowork-tour-notes.md` is a reference the participant can re-read in Module 1 if anything feels unfamiliar
- If the participant has used Cowork extensively before, this pre-work step is redundant. The customer's organisers decides whether to mark it as required or optional per cohort
