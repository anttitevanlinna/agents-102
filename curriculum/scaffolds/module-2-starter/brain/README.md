# brain/

Your compiled knowledge about one live challenge — the one you named in prework. Starts empty. Fills up during Module 2 once Claude reads `sources/` and extracts topic pages.

The brain is scoped to this challenge, not to your whole company. Everything on a brain page should help you think about the challenge better. If a claim doesn't touch the challenge, it doesn't belong here — at least not yet.

See the root `CLAUDE.md` for the topic-page shape.

Once Module 2 runs, this folder will have `index.md` and one file per major topic. Delete this README when the first topic pages land — the folder won't be empty anymore.
